# 2부 9장: 고객 여정 지도

## 핵심 요약

고객 여정 지도(Customer Journey Map)는 UX·UI 디자인에서 가장 쉽게 접할 수 있고 가장 광범위하게 쓰이는 매핑 프레임워크이다. 고객이 제품/서비스를 경험할 때 어떤 접점을 지니고 어떤 매력을 느끼는지, 그리고 어떻게 구매로 이어지는지의 과정을 시각적으로 표현한 그래프이다.

## 1. 고객 여정 지도의 개념

### 정의
- **고객 여정 지도(Customer Journey Map)**: 고객 또는 사용자가 겪는 서비스에 대한 경험을 브랜드에 대한 인식의 경험을 시간순으로 나타낸 시각적 매핑 도구
- User Journey Map, 고객 경험 지도(Customer Experience Map), 터치포인트 맵(Touch-point Map)이라고도 함

### 역사적 배경
- 1970년대 마케팅 연구에서의 고객의 구매 과정 분석에서 출발
- 1990년대 고객 행동을 본격적으로 분석하기 시작
- 2000년대 디지털 상거래 경험의 디자인에 본격 활용

### 고객 여정의 단계
- **인지(Awareness)** → **탐색(Consideration)** → **결정(Decision)** → **사용(Usage)** → **재사용/추천(Loyalty)**
- 단계별로 고객의 **행동, 감정, 터치포인트, 기회 요소** 등을 분석

## 2. 고객 여정 지도의 형식

### 구성 요소
| 구성 요소 | 설명 |
|------|------|
| **단계(Stage)** | 여정의 주요 단계 구분 |
| **행동(Action, Doing, Behavior)** | 각 단계에서의 고객 행동 |
| **생각(Thoughts)** | 고객의 사고 과정 |
| **감정(Feelings/Emotions)** | 감정 곡선으로 시각화 |
| **터치포인트(Touchpoint)** | 고객과 서비스의 접촉 지점 |
| **페인 포인트(Pain Point)** | 고객이 불편을 느끼는 지점 |
| **인사이트(Insight)** | 개선 기회와 핵심 발견 |
| **기회 요소** | 디자인 개선 방향 |

### 감정 곡선
- 고객의 감정 변화를 **곡선 그래프**로 시각화
- 긍정적/부정적 감정의 변화 추이를 한눈에 파악
- 페인 포인트와 **기회 영역**을 직관적으로 발견

## 3. 고객 여정 지도의 제작

### 제작 프로세스
1. **고객을 이해하고 설정**하기 (퍼소나 기반)
2. **고객의 여정 경험을 조사**하기
3. **여정을 구성하고 제작**하기 (페인포인트, 터치포인트 포함)
4. **메시지와 인사이트 도출**하기
5. **환경 정의**하기

### 제작 시 주의사항
- 고객 여정 지도는 **추상적이어선 안 되고 구체적**이어야 함
- 리서치 데이터에 근거하여 작성 (가정에 의존하지 않기)
- 현재 상태(As-Is)와 미래 상태(To-Be) 모두 작성 가능
- 하나의 퍼소나에 하나의 여정 지도 권장

### 여정 촬영과 공유
- 스마트폰으로 여정 사진 촬영하여 **인스타그램** 등에 공유
- 온라인 도구(스프레드시트, 디자인 툴) 활용하여 디지털 제작
- 벽면에 포스트잇으로 부착하여 **오프라인 협업**

### 터치포인트와 채널
- 고객이 서비스와 접촉하는 **모든 채널** 파악
- 온/오프라인 채널: SNS, 웹, 앱, 키오스크, 전화, TV 등
- **옴니채널** 관점에서 일관된 경험 설계

## 4. 고객 여정 지도의 유형

### 다양한 형태
- **패스/패턴(Path)**: UX 디자인 리서치의 이동 흐름
- **레일 유로프(Rail Europe)의 경험 지도(Experience Map)**: 여행 경험 전체를 매핑
- **Lens 여정 지도**: 고객의 시점에서 모든 단계를 모아 기회 포인트(Takeaways) 도출
- **스프레드시트형**: 엑셀/구글 시트로 제작한 체계적 여정 지도
- **UXD 프로젝트형**: 디자인 프로세스의 각 단계와 연결

### As-Is vs To-Be
- **As-Is**: 현재 고객 여정의 문제점 파악
- **To-Be**: 이상적인 미래 고객 여정 설계
- 두 지도를 비교하여 **개선 기회** 도출

## 5. 고객 여정 지도의 특징

### 핵심 특징
1. **미래(To-be)의 경험을 제시**: 이상적인 서비스 경험 설계
2. **퍼소나와 관련성**: 고객 여정 지도는 퍼소나를 기반으로 구체화
3. **As-Is 여정에서 어떤 불만을 발견**: 인터뷰와 관찰 데이터를 기반으로 페인 포인트 도출
4. **서비스 디자인과의 연관**: 시스템과 상호작용을 포함하는 전체적 관점
5. **사용자의 감정과 경험에 집중**: 기능이 아닌 감정적 여정에 초점

### 고객 여정 지도의 가치
- 프로젝트 전체 과정에서 **사용자 중심** 유지
- 이해관계자들과 **공통의 이해** 형성
- **페인 포인트** 발견과 **기회 영역** 도출의 핵심 도구

## 핵심 인사이트

1. 고객 여정 지도는 **퍼소나 기반**으로 구체적 사용자의 경험을 시각화
2. **감정 곡선**으로 긍정/부정 경험의 변화를 직관적으로 파악
3. **페인 포인트**와 **터치포인트** 분석이 디자인 개선의 핵심
4. **As-Is와 To-Be** 여정을 비교하여 개선 기회 도출
5. 고객 여정 지도는 **서비스 디자인**과 밀접하게 연결되며 전체적 관점 제공
