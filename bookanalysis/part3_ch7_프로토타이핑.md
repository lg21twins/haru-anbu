# 3부 7장: 프로토타이핑

## 핵심 요약

프로토타이핑(Prototyping)은 디자인 아이디어를 시각적으로 표현하고 실제 사용 환경에서 테스트할 수 있게 해주는 결과물을 만드는 과정이다. 프로토타이핑을 통해 디자이너와 개발자 간의 협업을 촉진하고, 사용자 피드백을 수집하여 사용성을 개선한다.

## 1. 프로토타이핑의 개념

### 정의
- 그리스어 'protos'(최초)와 'typos'(원본)의 합성어 'prototypon'에서 유래
- 완성된 제품 이전의 **시제품, 시작품(試作品)**을 의미
- 소프트웨어뿐 아니라 하드웨어에서도 중요한 역할

### 프로토타이핑의 중요성 — 사례
- 아이디오(IDEO)의 의료 기기 프로토타입: 스카치 테이프, 마커펜, 라이터, 빨래집게 등으로 **스터디 목업** 제작 → 회의에서 의견 반영된 프로토타입으로 발전
- **자포스(Zappos)**: 신발 사진을 찍어 웹사이트에 올리고 주문이 들어오면 직접 사서 배송 — 프로토타입으로 시장 검증

## 2. 프로토타이핑의 목적

### 여섯 가지 목적
1. **기능성, 사용성, 비주얼적 요소를 실제로 테스트** — 예상치 못한 문제점 발견
2. **협업(Collaboration)**: 디자이너와 개발자 간의 협업 촉진, 실시간 피드백
3. **설득(Persuasion)**: 추상적 개념을 구체적이고 시각적인 형태로 보여줌으로써 이해관계자 설득
4. **아이데이션 및 전개(Exploration)**: 창의적인 발상을 구체화하고 탐구
5. **사용자 테스트(User Testing)**: 디자인의 사용성을 평가하는 필수적 도구
6. **반복적 개선(Iteration)**: 초기 단계에서부터 다양한 버전의 프로토타입을 제작하고 피드백 반영

## 3. 프로토타이핑의 분류

### 수평 vs 수직 프로토타입
- **수평 프로토타입(Horizontal Prototype)**: 시스템의 여러 기능을 얕게 구현, 전반적인 사용자 경험을 초기에 테스트
- **수직 프로토타입(Vertical Prototype)**: 특정 기능이나 모듈을 깊이 있게 구현, 기술적 구현 가능성 확인

### 피델리티(Fidelity) 수준
| 수준 | 특징 |
|------|------|
| **Lo-Fi (Low-Fidelity)** | 손 스케치, 종이 프로토타입, 빠르고 저렴, 넓은 범위를 검토 |
| **Mid-Fi (Middle-Fidelity)** | Lo-Fi와 Hi-Fi 사이의 중간 단계 |
| **Hi-Fi (High-Fidelity)** | 완성도가 높아 충분한 사용 경험 전달, 시간과 비용 소요 |

### LF 프로토타입의 장점
- 구글 스프린트 개발한 제이크 냅: "더 나은 가짜를 보여주고 진짜라고 상상해보라"
- 손 스케치는 **작업 전환점**이 빠르고 효율적
- 페이퍼 프로토타입(Paper Prototype)은 빠르게 작업하고 테스트 가능한 LF 프로토타입

### HF 프로토타이핑 도구
- 피그마(Figma), 어도비 XD(Adobe XD), 프레이머(Framer) 등
- 2010년대까지는 전용 프로토타이핑 툴이 부재 → 파워포인트나 플래시로 제작

## 4. 분야별 프로토타이핑

### 시각 정보 디자인
- 2차원 평면에 배열하는 방식
- 피그마, 어도비 XD, 프로토파이(ProtoPie)로 제작

### 제품 디자인 — 스터디 목업
- **러프 목업(Rough Mock-up)**: LF 프로토타입, 스티로폼이나 골판지로 외형 제작
- **스터디 목업(Study Mock-up)**: 소프트 목업, 3D프린팅 이용
- **디자인 목업**: 최종 작업물과 실물을 구분할 수 없을 만큼 완성도 높음

### 경험 프로토타이핑
- 사용 상황, 행동, 느낌을 구현하는 프로토타이핑
- 구글 글라스를 안경에 스마트폰을 연결하여 초기 버전의 프로토타입으로 실험한 사례
- 닌텐도에서 의자의 초기 모형을 막대 모형에 사용해 사용자 경험을 테스트

### 앱 프로토타이핑
- 스마트폰과 태블릿 **애플리케이션 설계**에 중점
- 터치 인터랙션, 화면 전환, 반응형 레이아웃 등이 주요 설계 요소
- 제스처 기반 탐색(스와이프, 핀치) 등의 직관적인 터치 인터랙션 도입

### 공간 프로토타이핑
- 건축과 인테리어 디자인에서 주로 사용
- 3D 프로그램이나 실제 도구를 사용하여 공간을 시뮬레이션
- VR, AR, MR 등으로 공간 프로토타이핑 방식이 크게 변화
- 3D 프린팅이나 CNC(Computer Numerical Control) 가공 기술 활용

## 핵심 인사이트

1. 프로토타이핑은 **"빠르게 실패하고 빠르게 개선하는"** 반복적 과정의 핵심 도구
2. **수평(넓고 얕게) vs 수직(좁고 깊게)** 프로토타입을 프로젝트 단계에 맞게 선택
3. **Lo-Fi → Hi-Fi** 순서로 점진적으로 완성도를 높이되, LF 단계를 건너뛰지 말 것
4. 분야별(시각, 제품, 경험, 앱, 공간) 프로토타이핑의 **특성과 도구**가 다름을 이해
5. 프로토타이핑의 궁극적 목적은 **사용자 피드백**을 통한 디자인 개선
