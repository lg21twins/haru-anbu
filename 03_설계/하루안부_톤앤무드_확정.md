## 무드

### 가야 할 느낌 (DO)

- 가족에게 보내는 **안심 편지** 같은 앱
- **카페에서** 편하게 열어보는 앱
- **둥근 모서리** (border-radius: 16px+), 부드러운 그림자
- **여백 넉넉** — "덜 보여주는 용기"
- **라인 일러스트** 포인트 — 간결하고 따뜻
- 부드러운 말투 — "~하셨습니다", "~해보세요"
- **카드 기반** 레이아웃 — 정보를 카드 단위로 분리

### 피해야 할 느낌 (DON'T)

- 차가운 **병원 차트** 느낌
- 의료 **키오스크** 느낌
- 날카로운 직선, 각진 UI
- 빽빽한 데이터 나열
- 스톡 사진 과다
- 딱딱한 말투 — "~하십시오", "~바랍니다"
- 불안을 자극하는 빨간색 과다 사용
