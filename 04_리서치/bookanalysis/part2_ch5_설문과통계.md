# 2부 5장: 설문과 통계

## 핵심 요약

설문과 통계는 UX 리서치에서 정량적 데이터를 수집하고 분석하는 핵심 도구이다. 정성적 리서치만으로는 균형 있는 조사가 어렵기 때문에, 설문을 통한 통계적 분석이 반드시 병행되어야 한다. 이 장에서는 설문 설계, 통계의 기본 개념, 분석 방법, 연구에서 통계까지의 과정을 체계적으로 다룬다.

## 1. 설문, 정량 연구의 시작

### 설문의 특성
- 설문은 가장 흔하게 접하는 방법으로, **구조화된 질문**을 통해 데이터 수집
- **온라인 설문**과 **오프라인 설문**: Google Forms, Survey Monkey 등 도구 활용
- 설문은 **명확하고 간결**해야 하며, **모호한 표현** 지양

### 설문 문항의 유형
| 유형 | 설명 |
|------|------|
| **단일 선택(Radio Button)** | 하나만 선택 |
| **다중 선택(Check Box)** | 여러 개 선택 가능 |
| **콤보 박스(Combo Box)** | 드롭다운에서 선택 |
| **주관식** | 자유 기술 |

### 설문 작성 시 주의사항
- 설문 조사에서는 **최소화**하는 것이 핵심
- 개인 정보(나이, 성별 등)를 먼저 물으면 **편견** 유발 가능
- 질문 순서에 유의: 민감한 질문은 뒤로 배치
- **유도 질문** 지양, **이중 부정** 피하기
- 척도는 문항을 만들 때 연어적 뉘앙스가 중요

## 2. 통계에 대한 이해

### 통계의 기본 개념
- **통계학(Statistics)**: 'state(나라)의 산술(arithmetica)의 조합'에서 유래
- **모집단(Population)**: 전체 집단, **표본(Sample)**: 모집단에서 추출한 일부

### 표본 추출 (Sampling)
- **확률적 표본 추출**: 단순 무작위, 층화 표본, 군집 표본
- **비확률적 표본 추출**: 편의 표본(Convenience), 할당 표본(Quota), 의도 표본(Purpose)
- 대한민국 실질 남성의 평균 키를 조사하려면 약 **2,000명** 정도 필요
- 표본 크기는 **1,000명** 정도면 모집단을 대표 가능

### 데이터의 유형
| 대분류 | 소분류 | 특성 |
|------|------|------|
| **범주형(Categorical)** | 명목형(Nominal) | 분류만 가능 (성별, 혈액형) |
| | 순서형(Ordinal) | 순서 있음 (학력, 만족도) |
| **수치형(Numerical)** | 이산형(Discrete) | 정수값 (횟수, 인원) |
| | 연속형(Continuous) | 연속값 (키, 무게) |
| | 간격/등간(Interval) | 간격이 동일 (온도) |
| | 비율(Ratio) | 절대 영점 존재 (속도, 무게) |

### 통계 시각화 (그래프)
- **막대그래프**: 범주별 비교
- **원 그래프**: 비율/구성 표현
- **선 그래프**: 추세 변화
- **히스토그램**: 빈도 분포
- **레이더 차트**: 다차원 비교
- **산점도**: 변수 간 관계
- **누적 막대그래프**: 구성 요소의 누적

## 3. 통계의 분석 방법

### 기술통계 vs 추론통계
| 구분 | 설명 |
|------|------|
| **기술통계(Descriptive Statistics)** | 데이터의 특성을 요약·기술 |
| **추론통계(Inferential Statistics)** | 표본에서 모집단의 특성을 추론 |

### 모수통계 vs 비모수통계
| 모수통계(Parametric) | 비모수통계(Non-parametric) |
|------|------|
| 정규분포 가정 | 분포 가정 불필요 |
| t검정, 분산분석(ANOVA), 상관분석 | 카이제곱검정, Mann-Whitney U, Kruskal-Wallis |

### 기술통계의 주요 개념
- **평균(Mean)**: average(), 산술 평균
- **중앙값(Median)**: median(), 중앙에 위치한 값
- **최빈값(Mode)**: mode(), 가장 많이 나타나는 값
- **사분위수(Quartile)**: 25%, 50%, 75% 위치의 값
- **표준편차(SD)**: 데이터의 퍼짐 정도
- **분산(Variance)**: 표준편차의 제곱

### 정규분포와 표본 오차
- **정규분포 곡선(Normal Distribution Curve)**: 종 모양의 대칭 분포
- **가우스 분포(Gaussian Distribution)**라고도 함
- **68-95-99.7 법칙**: ±1σ(68%), ±2σ(95%), ±3σ(99.7%)
- **신뢰 수준(Confidence Level)**: 보통 95% 사용
- **표본 오차(Sampling Error)**: 표본과 모집단의 차이

## 4. 연구에서 통계까지의 과정

### 변수의 관계
- **독립변수(Independent Variable)**: 원인이 되는 변수
- **종속변수(Dependent Variable)**: 결과로 나타나는 변수
- **인과관계**: 원인과 결과의 관계
- **상관관계**: 두 변수 간의 관련성 (인과와 구분 필요)

### 가설 검증
- **영가설(Null Hypothesis, H₀)**: 차이가 없다는 가설
- **대립가설(Alternative Hypothesis, H₁)**: 차이가 있다는 가설
- **1종 오류(Type I Error)**: 영가설이 참인데 기각 (α 오류)
- **2종 오류(Type II Error)**: 영가설이 거짓인데 채택 (β 오류)
- **유의확률(p-value)**: 보통 0.05 미만이면 유의미 (*), 0.01 미만이면 매우 유의미 (**)

### 통계와 리서치의 결합
- 정성적 리서치 결과를 **통계적 방법으로 검증**
- **사전/사후 비교**, **A/B 테스트** 등에서 통계적 기법 적용
- 설문과 통계를 병행하여 **정성적+정량적** 리서치의 균형 확보

## 핵심 인사이트

1. 설문 설계 시 **최소화·명확성·비유도적 질문**이 핵심
2. **표본 추출** 방법에 따라 결과의 대표성이 달라짐
3. **기술통계**로 데이터 특성을 파악하고, **추론통계**로 모집단을 추론
4. **정규분포**와 **신뢰 수준(95%)** 이해가 통계 분석의 기초
5. **1종·2종 오류**와 **p-value**를 통한 가설 검증이 연구의 신뢰성을 결정
6. UX 디자이너는 통계에서 **무엇이 유의미한 것인지** 판단하는 능력이 필수
