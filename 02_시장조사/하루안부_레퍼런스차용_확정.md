## 4. Feature 차용 vs Design 차용 분리

### Feature 차용 (11개)

기능 로직, 시스템 동작, 데이터 흐름을 차용하는 항목:

- F-11 AI 자동 인수인계 리포트 (Birdie)
- F-10 AI 의료 행위 설명 (MyChart)
- F-7 AI 3단계 에스컬레이션 (채널톡 ALF)
- F-1 읽음 확인 + 재알림 (키즈노트)
- F-9 간병인 리뷰/별점 (케어닥)
- F-8 주간/월간 리포트 (Birdie)
- U-1 서클 기반 그룹 알림 (Life360)
- F-12 처치 동의 디지털화 (MyChart)
- U-5 카카오 연동 알림 (분당서울대병원)
- U-2 공동 작성자 (CaringBridge)
- F-4 가족 약물함 동기화 (Medisafe)

### Design 차용 (2개)

UI/UX 디자인 패턴, 화면 구성 방식을 차용하는 항목:

- D-2 카드 기반 리포트 UI (CaringBridge) — 날짜별 카드 레이아웃
- F-2 Lock Screen 위젯 (Medisafe) — 잠금화면 위젯 UI 패턴
