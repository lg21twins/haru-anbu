2.2  경쟁사 상세 분석
(1) 케어닥 — 간병인·요양보호사 매칭 플랫폼
항목
내용
유형
간병인·요양보호사 매칭 플랫폼 (B2C)
핵심 기능
AI 매칭, 24시간 긴급 매칭, 간병인 프로필·후기, 결제/정산 통합
서비스 범위
병원간병, 집 간병, 방문요양, 가사돌봄, 외출동행, 방문재활, 케어홈
강점
돌봄 업계 최초 양면 앱(보호자+종사자), 배상책임보험, 5단계 검증
약점
매칭에 집중 → 입소 후 실시간 소통/케어 현황 공유 기능 없음
케어 플랫폼과 비교
케어닥은 "사람 찾기", 우리는 "입소 후 소통" — 경쟁이 아닌 보완 관계

(2) 케어네이션 — AI 기반 간병·방문요양 매칭
항목
내용
유형
AI 기반 간병·방문요양 매칭 플랫폼 (B2C)
핵심 기능
AI 추천 매칭, 자격증 자동 분류, 신분 인증, 간병 동향 리포트
규모
앱 다운로드 200만+, 회원 69만, 매출 100억원+ (2023)
강점
6년 연속 소비자만족도 1위, AI 데이터 댐, 2025 서울형 R&D 선정
약점
매칭 중심 → 요양원 내 실시간 케어 소통 기능 없음
케어 플랫폼과 비교
케어네이션은 "매칭 + 구인", 우리는 "입소 후 케어 소통" — 서로 다른 단계

(3) 케어포 — 노인장기요양 기관 관리 프로그램 (B2B)
항목
내용
유형
노인장기요양 기관 관리 프로그램 (B2B SaaS)
핵심 기능
요양기관 업무관리, 급여기록, 비용명세서, 납부확인서, 공지사항
보호자 앱
"가족돌봄" 앱 — 급여기록 확인, 비용명세서, 공지 확인
강점
기관 운영 통합 관리, 보호자 앱 연동
약점
행정·비용 중심 → 실시간 케어 현황, AI 기능, 채팅 없음
케어 플랫폼과 비교
케어포는 "기관 행정 관리", 우리는 "케어 소통 + AI 가이드"

(4) 엔젤시스템 — 가장 직접적인 경쟁사
항목
내용
유형
장기요양시설 통합 사례관리 시스템 + 보호자 앱 (B2B)
핵심 기능
급여내역 확인, 어르신 하루 살펴보기, 소통 기능, 자동결제
보호자 앱
"굽은나무" — 어르신 하루 확인, 급여 내역, 소통, 자동 납부
강점
시설관리 + 보호자 소통 통합, PC + 모바일 지원
약점
AI 기능 없음, 단방향 정보 전달 중심, UI/UX 구식, 실시간 채팅 한계
가격
정원 기준 월사용료 + 세팅/교육 비용 (B2B 과금)
케어 플랫폼과 비교
엔젤시스템은 "시설 운영 + 단방향 공유", 우리는 "3자 양방향 소통 + AI"
엔젤시스템이 유일하게 "보호자 소통"을 제공하지만, 단방향 정보 전달 중심이며 AI 기능이 전혀 없습니다. 이것이 케어 플랫폼의 핵심 차별화 지점입니다.

강점
시설 탐색 특화, 복지용구 커머스, 보호자 커뮤니티

케어 플랫폼과 비교
또하나의가족은 "입소 전 탐색", 우리는 "입소 후 소통"
