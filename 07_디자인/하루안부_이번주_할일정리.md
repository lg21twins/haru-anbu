## 카테고리 2: 로고 제작

**추천 유형: 콤비네이션 마크** (심볼 + 워드마크, 상황에 따라 분리/결합)

### 만들어야 할 로고 세트

- [ ] **심볼** — 앱 아이콘으로 단독 사용 가능한 형태
- [ ] **워드마크** — "하루안부" 텍스트 로고
- [ ] **조합형** — 심볼 + 워드마크 (가로/세로 배치)
- [ ] **단색 버전** — 흰색, 검정
- [ ] **배경별 버전** — 밝은 배경용 / 어두운 배경용

### 로고 제작 시 고려사항

- **크기 대응**: 48×48px 앱 아이콘에서도 식별 가능. 디테일 많으면 뭉개짐
- **톤 일치**: 둥근 모서리, 부드러운 곡선. 날카로운 각/무거운 폰트 피하기
- **컬러**: 메인 안심 블루 #4A7FBF. 역할별 컬러 변형은 로고에 적용 X
- **이름 모티프**: 하루 = 해, 시간, 하루하루 / 안부 = 인사, 소통, 편지, 말풍선
