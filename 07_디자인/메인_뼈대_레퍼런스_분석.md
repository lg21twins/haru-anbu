## 2. 네비게이션 패턴

### Apple Health
- **하단 탭바**: 3~4탭 (Summary, Sharing, Browse + 돋보기)
- 탭 아이콘: 활성=파란색 채움, 비활성=회색 아웃라인
- 탭바 높이: ~50px, 흰 배경 + 상단 얇은 구분선
- **상단 헤더**: 좌측 `<` 뒤로 버튼(원형 회색 배경) + 중앙 타이틀 + 우측 액션(`+`, `...`)
- **섹션 헤더**: 좌측 큰 타이틀(22px Bold) + 우측 "Show All" 텍스트 링크(파란색)
- **모달 진입**: 바텀에서 슬라이드업, 상단에 X 닫기 버튼

### Apple Fitness
- **하단 탭바**: 3탭 (Summary, Workout, Sharing)
- 탭 아이콘: 활성=라임 그린 채움, 비활성=회색
- 탭바 높이: ~56px, 다크 배경(#0F0F0F)
- **상단 헤더**: `<` 뒤로(원형) + 날짜 타이틀 + 공유 아이콘
- **세그먼트 컨트롤**: 필 형태 탭 (Schedule / Trainers / Music), 활성=초록 배경+검정 텍스트

### 하루안부 적용 포인트
- **Apple Health 패턴 채용**: 라이트 모드 탭바, 섹션 헤더 패턴
- 탭 3~4개 (역할별 다름)
- 활성 탭 = `--primary` 컬러 (역할별)
- 섹션 헤더 = 좌측 타이틀 + 우측 "더보기" 링크

## 3. 카드 스타일

### Apple Health
- **기본 카드**: 흰 배경, radius 12-16px, 패딩 16px, 미세 그림자
- **추천 카드** (Get More From Health): 좌측 아이콘(컬러 원형 배경 안에) + 우측 텍스트 + "Get Started" 텍스트 버튼 + X 닫기
- **데이터 행**: 좌측 아이콘 + 중앙(제목+부제) + 우측 셰브론 `>`
- **상태 카드** (혈압 로그 등): 연한 컬러 틴트 배경(민트/라벤더/핑크), radius 12px, 프로그레스 표시
- **학습 카드** (Learn More): 좌측 일러스트(둥근) + 우측 제목+설명
- 카드 간 간격: 12-16px
- 카드와 화면 마진: 16px

### Apple Fitness
- **기본 카드**: 다크 그레이(#2D2D2D), radius 16-20px, 패딩 16-20px
- **콘텐츠 카드**: 썸네일 이미지 + 그라디언트 오버레이 + 제목 + 메타데이터
- **메트릭 카드**: 큰 숫자(32-48px, 노란/초록/시안) + 작은 라벨(12px, 회색)
- "NEW" 뱃지: 우상단, 작은 흰 텍스트

### 하루안부 적용 포인트
- **Apple Health 카드 구조 채용**: 흰 배경, radius 12px, 패딩 16px
- 데이터 행 패턴 → 환자 정보 목록, 체크리스트 항목
- 상태 카드(컬러 틴트) → 케어 완료율, 투약 상태
- 추천 카드 → AI 케어 가이드 진입점

## 7. 데이터 시각화

### Apple Health
- **프로그레스 링/도넛**: 수면 점수(84), 배경 회색 링 + 컬러 아크, 중앙에 큰 숫자
- **바 차트**: 세로 막대, 카테고리별 컬러

- **타임라인 도트**: 연속 원형 (빨강=기록됨, 빈 원=예정, 회색=미래)

### Apple Fitness
- **Activity Ring**: 동심원 3개 (Move/Exercise/Stand), 두꺼운 스트로크(8-12px)
- **시간 표시**: 큰 노란 숫자 HH:MM.SS (48-64px)
- **메트릭 칼럼**: 큰 숫자 + 작은 라벨 ("0 ACTIVE CAL")
- **세그먼트 바**: 가로 막대로 구간 표시
- **지도**: 다크 맵 + 노란 경로선

### 하루안부 적용 포인트
- **케어 완료율** → Apple Health 프로그레스 링/도넛 차트
- **투약 기록** → 타임라인 도트 패턴 (완료=초록, 미완료=빈 원)
- **대시보드 통계** → Fitness 메트릭 칼럼 (숫자 강조 + 라벨)
- **일과표** → 수평 프로그레스 바 + 시간 라벨

- **투약 추적**: 약 캡슐 비주얼(모양+색상 커스텀), 약물명+용량+빈도 카드, 스케줄 시각화(시간+수량), 기록 = 원형 아이콘+타임스탬프+상태
- **기분 체크**: 이모지/텍스트 선택 기반
- **설문**: 질문지 형태 온보딩, 점수 시각화(파이/도넛), 위험 레벨 뱃지(Minimal/Mild/Moderate/Severe)
- **수면**: 수면 점수(큰 숫자+컬러 링), 지속시간, 취침/중단 메트릭
- **공유**: 건강 데이터 카테고리 토글 on/off, 개인정보 보호 안내
- **PDF 내보내기**: "Export PDF" 버튼

### 하루안부 적용 포인트
- **케어 체크리스트** → 투약 추적 패턴 (체크+시간+상태)
- **AI 리포트** → Health 결과 화면 패턴 (요약+다음 단계)
- **대시보드** → Health Summary 패턴 (카드 나열 + 프로그레스)
- **SOS** → Alert 패턴 강화 (빨간 강조)

## 14. 전체 디자인 언어 특징

### Apple Health의 핵심
1. **데이터 중심이지만 인간적** — 큰 숫자 + 따뜻한 언어 + 교육적 지원
2. **카테고리별 컬러 코딩** — 한눈에 정보 구분 가능
3. **프로그레시브 디스클로저** — 핵심만 보여주고 "Show All"로 확장
4. **일러스트 활용** — 스톡이 아닌 커스텀 일러스트로 친근함
5. **프라이버시 강조** — 암호화, 권한 기반 공유 안내
6. **최소한의 시각 노이즈** — 깨끗한 흰 배경, 미세 구분선, 넉넉한 여백
7. **iOS 네이티브 일관성** — 시스템 폰트, 시스템 컬러, 표준 컴포넌트
8. **데이터의 서사화** — "You averaged 1 day 14hr..." 숫자를 문장으로

### Apple Fitness의 핵심
1. **다크 퍼스트** — 시각적 편안함, 밝은 액센트로 집중도
2. **성과 중심 메트릭** — 큰 볼드 숫자, 밝은 컬러
3. **커스터마이제이션** — 플랜 빌더, 음악/트레이너 선택
4. **소셜 통합** — 친구 아바타, 활동 공유, 스트릭

### 하루안부에 가져갈 것
- Health의 **라이트/깨끗/여백/카드** 기반 + **프로그레시브 디스클로저**
- Health의 **카테고리별 컬러** → 역할별 컬러(블루/그린/베이지)
- Health의 **서사화** → AI 리포트에서 "오늘 어머님은 이렇게 보내셨어요"
- Fitness의 **프로그레스 링** → 케어 완료율 시각화
- Fitness의 **스케줄 관리 UI** → 의료진 일정 관리
