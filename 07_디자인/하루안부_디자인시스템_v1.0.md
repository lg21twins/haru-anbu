## 2. 타이포그래피

**서체: Pretendard Variable** (CDN: orioncactus/pretendard)

```css
font-family: 'Pretendard Variable', 'Pretendard', -apple-system, 'Helvetica Neue', sans-serif;
```

### 타입 스케일

| 토큰 | 크기 | 굵기 | 사용처 |
|------|------|------|--------|
| `--text-display` | 32px | SemiBold 600 | 화면 대제목 (홈 인사, 큰 숫자) |
| `--text-title` | 22px | SemiBold 600 | 섹션 제목, 모달 헤더 |
| `--text-headline` | 17px | SemiBold 600 | 카드 제목, 리스트 행 제목 |
| `--text-body` | 16px | Regular 400 | 일반 본문 |
| `--text-body-large` | 18px | Regular 400 | 환자 전용 본문 (고령자 가독성) |
| `--text-callout` | 14px | Regular 400 | 보조 정보, 부제목 |
| `--text-caption` | 13px | Regular 400 | 캡션, 타임스탬프 |
| `--text-mini` | 11px | Regular 400 | 최소 텍스트, 뱃지 |

### 자간/행간

| 토큰 | 값 | 사용처 |
|------|-----|--------|

## 5. 그림자 (Shadows)

| 토큰 | 값 | 사용처 |
|------|-----|--------|
| `--shadow-sm` | `0 1px 4px rgba(0,0,0,0.06), 0 0 1px rgba(0,0,0,0.04)` | 일반 카드 |
| `--shadow-md` | `0 4px 16px rgba(0,0,0,0.08), 0 0 1px rgba(0,0,0,0.04)` | 중요 카드, 버튼 그룹 |
| `--shadow-lg` | `0 8px 28px rgba(0,0,0,0.12), 0 0 1px rgba(0,0,0,0.04)` | 히어로 카드, 모달 |

## 7. 버튼 컴포넌트

### 종류

| 클래스 | 배경 | 텍스트 | 사용처 |
|--------|------|--------|--------|
| `.btn-primary` | `--primary` (역할 컬러) | 흰색 | 주요 CTA |
| `.btn-secondary` | `--bg` (라이트) | 기본 텍스트 | 보조 액션 |
| `.btn-destructive` | `#DC2626` (빨강) | 흰색 | 삭제, 위험 액션 |
| `.btn-ghost` | 투명 | `--primary` | 인라인 링크형 버튼 |

### 수식자

| 클래스 | 효과 |
|--------|------|
| `.btn-full` | width: 100% |
| `.btn-sm` | height: 36px, 소형 |
| `.btn-lg` | height: 54px, 대형 |

### 인터랙션
- `:active` → `opacity: 0.8` + `scale(0.98)` (iOS 탭 피드백)
- 전환 시간: `150ms ease`

## 8. 카드 컴포넌트

## 9. 헤더 & 탭바

### 페이지 헤더 (`.header`)
```
position: sticky, top: 0
height: 44px
배경: --bg-gradient (반투명 효과)
backdrop-filter: blur(12px)
구성: 좌(뒤로가기 or 아이콘) + 중앙(제목) + 우(액션)
```

### 탭바 (`.tabbar`)
```
position: fixed, bottom: 0
height: ~84px (패딩 포함)
배경: rgba(255,255,255,0.95)
backdrop-filter: blur(20px)
border-top: 0.5px solid --divider
구성: 아이콘 + 텍스트 라벨 (항상 병기)

비활성 탭: --text-tertiary
```

## 14. 애니메이션

### 페이지 진입 (fadeInUp)
```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(16px); }
  to   { opacity: 1; transform: translateY(0); }
}
animation: fadeInUp 300ms ease forwards;
```

### 전환 토큰

| 토큰 | 값 | 사용처 |
|------|-----|--------|
| `--transition-fast` | 150ms ease | 버튼 탭, 토글 |
| `--transition-normal` | 250ms ease | 카드, 모달 진입 |
| `--transition-slow` | 350ms ease | 화면 전환, 슬라이더 |
