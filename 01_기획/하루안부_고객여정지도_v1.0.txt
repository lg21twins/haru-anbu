   01   고객 여정 지도 개요
고객 여정 지도는 사용자가 서비스를 인지하고, 사용하고, 이탈하기까지의 전 과정을 시각화한 것입니다. 본 문서에서는 1차 퍼소나인 김미영(보호자)의 관점에서 As-Is(현재)와 To-Be(하루안부 도입 후) 여정을 비교 분석합니다.
분석 대상: 김미영 (52세, 직장인 딸, 치매 초기 어머니를 요양원에 모심)
   02   As-Is 고객 여정 지도 (현재 상태)
하루안부 도입 전, 김미영이 어머니의 요양원 생활에서 겪는 현재 경험입니다.
Stage 1: 입소 결정
항목
내용
행동
요양원 검색, 지인 추천, 3~5곳 방문 비교, 입소 계약
생각
'어떤 곳이 엄마한테 좋을까', '내가 직접 돌봐야 하는 거 아닌가'
감정
😰 불안 · 죄책감 (감정 점수: 2/5)
터치포인트
네이버 검색, 장기요양 등급 판정 기관, 요양원 현장 방문, 지인 추천
페인포인트
요양원 비교 기준 불명확, 비용 구조 복잡, 죄책감으로 인한 의사결정 지연
Stage 2: 일상 (출근/퇴근)
항목
내용
행동
출근 중 요양원 공지 확인, 퇴근 후 전화 시도 (평균 3회), 주말 면회
생각
'오늘 엄마 밥은 잘 드셨을까', '전화 왜 안 받지'
감정
😟 걱정 · 답답함 (감정 점수: 2.5/5)
터치포인트
카카오톡 단톡방(공지만), 요양원 대표번호 전화, 월 1회 면회
페인포인트
전화 연결률 60%, 간호사 바쁜 느낌으로 통화 짧음, 실시간 상태 확인 불가
Stage 3: 궁금할 때
항목
내용
행동
인터넷 검색(치매 정보), 카페·블로그 읽기, 지인에게 물어봄
생각
'이 증상이 심해지는 건가', '어디서 정확한 정보를 얻지'
감정
😧 불안 · 혼란 (감정 점수: 2/5)
터치포인트
네이버 카페, 블로그, 유튜브, 지인 구전
페인포인트
정보 출처 불명확, 과잉/부정확 정보로 불안 가중, 전문가 상담 접근 어려움
Stage 4: 긴급 상황
항목
내용
행동
요양원에서 전화 옴 → 상황 파악 → 병원 이동 → 사후 처리
생각
'큰일 난 건 아닌가', '왜 더 빨리 연락 안 했지'
감정
😱 공포 · 분노 (감정 점수: 1/5)
터치포인트
요양원 전화 (일방적 통보), 119, 병원 응급실
페인포인트
알림 지연, 상황 정보 부족, 의료 기록 공유 안 됨, 연락 체계 혼란
Stage 5: 비용/행정
항목
내용
행동
월 비용 계산, 장기요양보험 서류, 등급 갱신 준비
생각
'비용이 얼마나 더 들까', '보험 혜택을 제대로 받고 있나'
감정
😩 부담 · 피로 (감정 점수: 2/5)
터치포인트
국민건강보험공단, 요양원 행정실, 은행
페인포인트
비용 구조 불투명, 서류 복잡, 지원금 정보 산재, 시간 부족

   03   To-Be 고객 여정 지도 (하루안부 도입 후)
하루안부 서비스 도입 후, 동일한 김미영의 여정이 어떻게 변화하는지 보여줍니다.
Stage 1: 입소 결정 → 하루안부와 함께
항목
내용
행동
하루안부 AI 케어가이드 '요양원 찾기' 모드로 조건별 검색, 비용 시뮬레이션
생각
'이 앱에서 비교가 되네', '비용도 미리 계산해볼 수 있구나'
감정
🙂 안도 · 정보 확보 (감정 점수: 3.5/5)
터치포인트
하루안부 앱 → AI 케어가이드 (요양원 검색 + 비용 계산기 모드)
개선 포인트
비교 기준 명확화, 비용 시뮬레이션으로 불확실성 해소
Stage 2: 일상 → 대시보드 + AI 리포트
항목
내용
행동
출근길에 대시보드로 어머니 상태 확인. 저녁에 AI 일일 리포트 수신. 채팅으로 간호사 소통
생각
'아, 오늘 밥 잘 드셨네', '혈압도 안정적이고 다행이다'
감정
😊 안심 · 연결감 (감정 점수: 4/5)
터치포인트
하루안부 대시보드, AI 일일 리포트 알림, 의료진 채팅
개선 포인트
전화 불필요 → 실시간 확인. 감정 점수 2.5 → 4로 상승
Stage 3: 궁금할 때 → AI 케어가이드
항목
내용
행동
하루안부 AI 케어가이드 '질병 백과' 모드로 치매 정보 검색, '감정 지지' 모드로 상담
생각
'전문 정보가 정리돼 있으니 안심', '나만 힘든 게 아니었구나'
감정
😌 안정 · 위로 (감정 점수: 3.5/5)
터치포인트
AI 케어가이드 (질병 백과 + 감정 지지 모드)
개선 포인트
신뢰 가능한 정보원 확보, 감정적 지지까지 통합 제공
Stage 4: 긴급 상황 → SOS + 실시간 알림
항목
내용
행동
SOS 실시간 알림 수신 → 상황 상세 확인 → 앱 내 의료진 소통 → 처방전/의료기록 확인
생각
'바로 알림이 오니 빠르게 대처할 수 있다', '의료 기록도 바로 보이네'
감정
😮 긴장하지만 통제감 있음 (감정 점수: 3/5)
터치포인트
SOS 알림, 앱 내 긴급 채팅, 처방전/의료기록 공유
개선 포인트
알림 지연 해소, 정보 공유로 불안 감소. 감정 점수 1 → 3으로 상승
Stage 5: 비용/행정 → 통합 관리
항목
내용
행동
AI 케어가이드 '비용 계산기' 모드로 월 비용 시뮬레이션, '돌봄 제도' 모드로 지원금 안내
생각
'한 앱에서 다 되니 편하다', '이런 지원금이 있었네'
감정
🙂 편안 · 파악 완료 (감정 점수: 3.5/5)
터치포인트
AI 케어가이드 (비용 계산기 + 돌봄 제도 모드)
개선 포인트
비용 투명화, 지원금 안내 자동화. 감정 점수 2 → 3.5로 상승

   04   핵심 인사이트
1. 가장 빈번한 여정(일상)에서 가장 큰 감정 개선 효과 — 대시보드와 AI 리포트가 핵심
2. 가장 극단적 감정(긴급 상황)에서 +2점 개선 — SOS 알림이 '공포'를 '통제 가능'으로 전환
3. 정보 불안(궁금할 때)을 AI 케어가이드로 해소 — 신뢰할 수 있는 단일 채널 제공
4. 보호자의 죄책감 완화 — '연결되어 있다'는 느낌이 죄책감을 줄여줌
