────────────────────────────────────────────────
한 줄 컨셉
────────────────────────────────────────────────

"요양원판 토스 + ChatGPT"
— 매일의 안심을 전하는 AI 케어 소통 플랫폼

요양원에 가족을 맡긴 보호자가 전화 없이도 오늘 하루를 알 수 있도록,
AI가 매일 케어 리포트를 자동 생성하고 보호자·의료진·환자 3자를 하나의 앱으로 연결합니다.


────────────────────────────────────────────────
핵심 문제 정의
────────────────────────────────────────────────

"요양원에 가족을 맡긴 보호자는 오늘 엄마가 밥을 드셨는지조차 알 수 없고,
 의료진은 보호자 전화 응대에 하루 2시간을 빼앗기며,
 환자는 가족과 디지털로 연결될 수단이 없다."

→ 정보 단절로 인한 불안 → 연결을 통한 안심 으로 전환하는 것이 하루안부의 미션.


────────────────────────────────────────────────
타겟 사용자 3종
────────────────────────────────────────────────

① 보호자 (Primary ★★★)
   김미영, 52세, 서울 직장인
   "퇴근 후 전화해도 안 받을 때가 제일 불안합니다."
   니즈: 실시간 케어 현황 확인 + 긴급 알림 + 요양 정보
   규모: 장기요양 인정자 가족 추정 200만 명+

② 의료진 (Secondary ★★★)
   박지현, 34세, 요양원 간호사 7년차
   "하루에 보호자 전화 10통, 다 같은 질문이에요."
   니즈: 체크리스트 기록 간소화 + 보호자 소통 자동화 + 인수인계 효율화
   규모: 요양보호사 등록자 약 57만 명

③ 환자 (Served ★★☆)
   김순자, 78세, 장기요양 3등급
   "딸이 보고 싶은데... 오늘 뭐 하는지 알면 좋겠어."
   니즈: 큰 글자 UI + 일과 확인 + 가족 연결 + 원터치 도움 요청
   규모: 장기요양 이용자 117만 명


────────────────────────────────────────────────
핵심 기능 TOP 3
────────────────────────────────────────────────

① AI 일일 케어 리포트 (핵심)
   의료진 체크리스트 입력 → 매일 저녁 AI(Claude API)가 자동 요약
   → 보호자에게 "오늘 어머님은 이렇게 보내셨어요" 리포트 발송
   전화 없이도 오늘 하루를 알 수 있다.

② 5개 모드 AI 케어가이드 (핵심 차별화)
   제도안내 / 건강정보 / 감정지원 / 요양원찾기 / 요양비계산
   "어디에 물어봐야 할지 모르는" 보호자를 위한 원스톱 정보 채널.
   3단계 에스컬레이션으로 미해결 시 의료진에게 자동 연결.

③ SOS 긴급 알림 + 실시간 소통
   의료진 SOS 버튼 → 보호자 30초 이내 푸시 알림
   + 케어 체크리스트 실시간 동기화 (의료진 입력 → 보호자 대시보드 3초 내 반영)
   + 1:1 채팅으로 보호자·의료진 비동기 소통


────────────────────────────────────────────────
핵심 차별점 3가지
────────────────────────────────────────────────

1. 유일한 3자 실시간 소통
   보호자 ↔ 의료진 ↔ 환자 모두 연결.
   국내 경쟁사는 최대 2자 연결 (매칭 또는 시설↔보호자 단방향).
   입소 후 실시간 소통 영역: 국내 완전 공백 = 블루오션.

2. AI가 매일 자동 생성하는 케어 리포트
   전화 없이도 오늘 하루를 알 수 있다.
   Claude API 기반, 보호자 눈높이의 자연어 요약.
   현재 국내 시장에 유사 서비스 없음.

3. 원스톱 케어 정보 플랫폼 (5개 모드 AI 가이드)
   요양원 찾기부터 비용 계산, 건강 정보, 감정 지원까지 하나의 앱에서.
   "어디서 정보를 얻어야 할지 모르는" 보호자를 위한 단일 창구.
