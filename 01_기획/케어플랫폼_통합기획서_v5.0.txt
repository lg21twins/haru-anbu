01  문제 정의
요양시설 이용 노인은 매년 빠르게 증가하고 있지만, 보호자와 요양시설을 실시간으로 연결하는 소통 수단이 존재하지 않습니다. 가족은 오늘 부모님이 밥을 드셨는지조차 알 수 없고, 긴급 상황 발생 시 골든타임을 놓치는 피해가 반복됩니다. 또한 처방전이나 진료기록을 보호자가 직접 확인할 방법이 없고, 요양원에 대해 잘 모르는 보호자가 건강·요양 정보를 창구도 부재합니다.

02  서비스 개요
보호자·의료진(의사/간호사)·환자를 하나의 앱으로 연결해 케어 현황을 실시간으로 공유하고, AI 케어 가이드로 진료정보를 안내하며, 처방전·진료기록까지 투명하게 열람할 수 있는 AI 케어 소통 플랫폼입니다.

의사, 간호사, 요양보호사. PC에서 케어 기록, 처방전 작성, 보호자 소통 효율화가 주요 니즈.
요양시설 입소 노인. 안전하고 질 높은 케어 서비스, 가족 연결, 일과 확인이 주요 니즈.

플랫폼 전략: 웹 우선 개발
의료진는 주로 PC에서 작업합니다. 처방전 작성, 진료기록 확인 등을 데스크톱 컴퓨터로 수행하므로, 웹을 먼저 개발하고 모바일 반응형 + PWA로 확장하는 전략을 채택합니다.
	•	의료진: PC 웹 대시보드 최적화 (키보드 단축키, 넓은 화면 활용)
	•	보호자: 모바일 반응형 웹 → PWA 설치
	•	환자: 모바일/태블릿 반응형 웹 (큰 글자 + 단순 UI)

03  핵심 기능 (Level 1)
공모전 제출 시 실제로 동작하는 9개 기능. 각 기능은 문제 → 해결 구조로 설계.

01 케어 체크리스트
문제: 의료진 케어 수행 여부를 구두로만 전달 → 보호자 확인 불가
해결: 식사·투약·위생·활동 체크리스트 앱 기능 → 보호자 실시간 열람
예시: "오전 9:00 투약 완료 ✓ 점심 식사량 80% ✓"
02 SOS 긴급 알림
문제: 응급 상황 발생 시 보호자 연락 지연 → 골든타임 손실
해결: 의료진 SOS 버튼 → 보호자 즉시 알림 → 미응답 시 자동 재알림
예시: "[긴급] 101호 김○○ 어르신 낙상 의심. 즉시 확인 움컜"
03 1:1 채팅 + AI 일일 리포트
문제: 보호자-의료진 소통 창구 없음 → 정보 단절
해결: 채팅 + 매일 저녁 AI가 하루 케어 요약 리포트 자동 발송 (Claude API)
04 AI 케어 가이드 (핵심 차별화)
문제: 요양원에 대해 잘 모르는 보호자가 건강·요양 정보를 창구도 부재
해결: AI 케어 가이드 — 진료정보 백과사전 + 감정 지원(서브)
	•	메인: "장기진료등급 신청은 어떻게 하나요?" → 절차·서류·기간 상세 안내
	•	메인: "치매 초기 증상이 뭐예요?" → 노인성 질환 기초 정보 제공
	•	서브: 대화 흐름에서 감정적 신호 감지 시 자연스럽게 공감·위로 모드로 전환
05 시간표 (★ NEW)
문제: 환자의 하루 일과를 보호자가 파악할 수 없고, 의료진의 케어 스케줄이 체계적으로 관리되지 않음
해결: 환자 일과표(환자·보호자 열람) + 의료진 근무/케어 스케줄(PC 웹)
	•	환자: 모바일/태블릿에서 큰 글자로 오늘 일과 확인
	•	보호자: 환자의 일과표 열람하여 어떤 활동을 하는지 파악
	•	의료진: PC에서 오늘 담당 환자 목록 + 케어 타임라인 한눈에 파악
06 처방전·진료기록 + 의료 타임라인 (★ NEW)
문제: 보호자가 환자의 처방전이나 진료기록을 직접 확인할 방법이 없음
해결: 의료진가 PC에서 입력·관리, 보호자·환자는 권한에 따라 열람
	•	의료진: 처방전 입력(약명, 용량, 투약시간) + 진료기록 입력(진단명, 소견, 추후계획)
	•	보호자: 처방전 확인 + 진료기록 요약 열람
	•	환자: 본인 처방전 확인 (큰 글자 UI)
	•	보안: Supabase RLS로 역할별 접근 범위 강제
▶ 의료 타임라인 (진료·처방·꺀사 통합 뷰)
	•	진료기록, 처방 변경, 꺀사 기록을 시간순 타임라인으로 통합 표시
	•	보호자: 환자의 최근 의료 이력을 스크롤 한 번으로 파악 (':>': 3/14 정기검진 → 3/12 처방 변경 → 3/10 현류검사)
	•	의료진: PC에서 환자별 전체 의료 이력 타임라인 joao + 젼 기록 추가
	•	F-08(시간표)과 F-09(처방전·진료기록) 데이터를 하나의 흐름으로 시각화

07 AI 케어 가이드 확장 — 원스톱 케어 정보 플랫폼
기존 AI 케어 가이드(백과사전 + 감정 지원)에 3개 실용 모드를 추가하여, 보호자가 겪는 "물어볼 데가 없는" 모든 문제를 AI 챗봇 하나로 해결합니다.
요양원 찾기 (NEW)
급하게 요양원을 알아봐야 할 때, 대화를 통해 조건(지역, 등급, 유형, 예산)을 파악하고 공공 데이터 기반으로 맞춤 요양원을 검색합니다. 기존 서비스(또하나의가족)는 필터 검색이지만, 우리는 "뭘 검색해야 하는지조차 모르는" 보호자를 위한 대화형 검색입니다.
요양보호사 안내 (NEW)
요양보호사를 구하는 방법을 안내하고, 전문 매칭 서비스(케어닥, 케어네이션)로 바로 연결합니다. 직접 매칭을 하지 않고 "정보 안내 + 외부 연결"로 포지셔닝하여 경쟁이 아닌 생태계 협력 관계를 구축합니다.
요양비 계산 (NEW)
등급·감경·비급여 등 복합 조건을 반영한 비용 시뮬레이션을 제공합니다. 시설 vs 재가 비용 비교, 기초수급자/차상위 감경 반영, 비급여 항목 포함 총비용 산출까지 대화형으로 안내합니다. 현재 시장에 이런 "종합 비용 계산" 서비스는 존재하지 않습니다.
