12  정리
질문
답
뭘 만드나?
요양원 보호자·의료진·환자를 하나의 앱으로 연결하는 AI 케어 소통 플랫폼 "하루안부"
왜 만드나?
입소 후 보호자가 케어 현황을 알 수 없고, 소통할 수 없고, 정보를 구할 수 없는 문제 해결
뭐가 다른가?
1) 3자 실시간 소통 (국내 최초) 2) AI 케어 가이드 5개 모드 + 3단계 에스컬레이션 3) AI 자동 인수인계 + 일일/주간/월간 리포트 4) AI 의료 행위 설명 + 처치 동의 디지털화
어떻게 만드나?
React + Supabase + LLM API 전 과정 AI 활용으로 3인 팀 생산성 극대화
일정은?
W1-7: 기획+디자인 → W8: 중간발표 W9-14: 구현+정교화 → W15: 최종 시연
UX 프로세스는?
교과서 8단계 모델 기반 발견→정의→구조→디자인→프로토→테스트→개선→배포 현재 4단계(디자인) 진행 중


가족은 알고 싶다.  의료진은 기록을 줄이고 싶다.  환자는 연결되고 싶다.
