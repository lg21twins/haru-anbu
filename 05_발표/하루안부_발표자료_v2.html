Insight Conclusion

보호자는 전화해야만
오늘을 알 수 있었을까요?

요양원 소통 문제는 오래됐지만, 아무도 해결하지 않았습니다.
정보 단절은 불안을 만들고, 불안은 반복 전화가 됩니다.

하루안부는 번거로운 전화보다 실시간 연결이 쌓이도록,
케어 소통을 따뜻하고 자연스럽게 바꿔줍니다.

매일 하루, 매일 안부.
하루안부는 그 연결을 멈추지 않습니다.

SWOT Analysis
Strength
강점
(1) 국내 유일 3자 실시간 연결 — 보호자·의료진·환자
   → 입소 후 소통 영역 완전 공백 = 블루오션
(2) Claude API 기반 AI 일일 케어 리포트 자동 생성
   → 전화 없이도 오늘 하루를 알 수 있다
(3) Supabase Realtime — 3초 이내 체크리스트 반영
Weakness
약점
(1) 초기 단계 — 시설 영업력 및 레퍼런스 부재
(2) Claude API 비용 — 규모 확장 시 AI 운영 비용 증가
(3) 요양원 디지털 전환 저항 — 고령 의료진 적응 필요
(4) 팀 규모 소규모 3인 — 빠른 기능 확장 한계
Opportunity
기회
(1) 2025년 초고령사회 진입 — 요양·케어 수요 폭발 성장
   → 장기요양 이용자 매년 10만+ 증가 추세
(2) 국내 입소 후 실시간 소통 서비스 완전 공백
(3) AI 헬스케어 규제 완화 + 정부 디지털 전환 지원금
Threat
위협
(1) 카카오·네이버 등 대형 IT의 케어 시장 진입 가능성
(2) 의료정보 개인정보보호 규제 강화 리스크
(3) 케어닥 등 기존 플랫폼의 기능 확장 대응
(4) 요양원 시설 측 디지털 도입 거부감
SO 전략
초고령사회 블루오션에서 3자 실시간 연결 + AI 리포트의 유일한 조합으로 선점  |  요양원 직접 영업 → 의료진이 보호자에게 앱 안내 → 바이럴 확산
